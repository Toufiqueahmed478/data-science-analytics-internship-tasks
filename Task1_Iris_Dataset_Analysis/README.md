# Data Science & Analytics Internship Tasks

This repository contains completed internship tasks for DevelopersHub Corporation.

## Task 1: Exploring and Visualizing a Simple Dataset

### Objective
The objective of this task is to understand how to read, summarize, and visualize a simple dataset using Python.

### Dataset
Iris Dataset, loaded using the seaborn library.

### Approach
- Loaded the Iris dataset using seaborn.
- Inspected the dataset using shape, columns, and head.
- Checked basic summary statistics.
- Created visualizations using matplotlib and seaborn.
- Used scatter plot, histogram, and box plot to understand relationships, distributions, and spread.

### Results and Insights
- Petal length and petal width clearly help separate iris species.
- Setosa is clearly different from versicolor and virginica.
- Histogram helps understand feature distribution.
- Box plot helps compare spread and possible outliers between species.
